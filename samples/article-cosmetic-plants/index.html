Magazine Layout with Interactive Data

Adaptation of a print magazine article implemented as a web page with a responsive grid layout and sortable tabular data visualization. The article compares plants used in cosmetics production for fictional company “Virna & Nuris”. Assets include a functioning version of the article, CSS stylesheet, JavaScript module, and detailed implementation document.

Description#

Web Design/Development

Component Focused Implementation

My approach was to design an article template for a real-world ecommerce application. My focus was to divide the page into reusable components that could also be overridden and extended.

Interactive & Responsive Data Table

A data canvas was designed to be a module within the larger article framework. The data canvas includes a data visualization, interaction instructions, and interactive elements.

For this article example, sort buttons are inserted into a standard table to create a table with sortable columns. A loader is displayed while the table is being converted into a sortable table. If conversion is not possible or an error occurs, the unmodified table is displayed and the elements indicating the table is interactive are hidden.

For large screens and print, the table is shown in a standard tabular format. For small screens, the table is disassembled and the columns are converted into blocks. Depending on viewport dimensions, the blocks are displayed in a horizontal scrolling container or as a grid.

Element Sizes Informed by User Agent Font Size

The article has a multi-column or single column layout based on the user's font size and the user agent dimensions.

Element sizes and media queries are based on the user's font size. All containers are implemented to break words if necessary to prevent horizontal scroll.

Box properties are calculated using font size and viewport dimensions. This ensures that dimensions do not become exaggerated due to large fonts on small screens.

Generated Color Palette

The HSL color palette consists of 2 branded colors and a gray neutral. Each color has 5 contrast levels in relation to the background color: highest, high, medium, low, and lowest. The highest contrast color is used as a base to generate the remaining colors by gradually increasing the color lightness property.

Color Reversal for Dark Mode

All surfaces have light background colors and contrasting dark text. When user's preferred color scheme is dark, the colors are reversed such that the surfaces have dark background colors and the text is lighter in contrast.

Responsive Graphical Features

Raster images selected using pixel density and device pixel ratio. Print display is optimized by removing extraneous elements and spacing, and shadows.

Masking for Dynamic Graphics Color

Image masking is used to make black logos and icons the same color as their container's text.

Accessibility Features

Touch targets are implemented with a minimum size. Speed of animations is reduced based on user’s motion preference. Sortable data visualizations use aria-sort attribute to announce ascending versus descending sort.

In the content, abbreviations that are often misread by screen readers are structured to include the full word as visually hidden and the abbreviation as aria hidden so the full word is read and the abbreviation is ignored.
