Brochure Realtor Website

A brochure website template that provides a professional online presence, credibility, and brand awareness for a local real estate company. The static content includes company background information, recent sales, and contact details. Responsive sizing and color based on user agent configurations. Interactive elements are implemented such that scripting is not required.

Description#

Web Design/Development

Minimal Aesthetics for Clean UX/UI

A flat design and muted color scheme create a professional user interface. Large scale elements and copious whitespace establish a visual hierarchy and the appearance of abundant content. Visual interest is applied with varying fonts, text size and case, borders, and shading.

Dynamic Color with HSL Palette & Masks

A CSS generated HSL color palette consists of 2 branded colors and a gray neutral. Each color has 5 contrast levels in relation to the background color: highest, high, medium, low, and lowest. The highest contrast color is used as a base to generate the remaining colors by gradually increasing the color lightness property.

Image masks are used to colorize black & white logos and icons based on the color palette.

User Color Preferences

The user interface is designed with light surfaces and dark contrasting text. In Dark Mode, the full color palette is reversed, creating dark surfaces and light contrasting text.

In Forced Colors Mode, containers that use backgrounds to provide a visual bounding box are configured with borders. System colors are used to define the colors applied to elements that would be non-functional with their color removed.

Accessibility & Usability

Focus on accessibility and usability for keyboard users and screen readers.

Care is taken not to include text characters in decorative pseudo elements, opting instead for adding characters to hidden DOM elements or fashioning shapes with CSS boxes. This ensures that screen readers using the Accessibility Tree as an alternate version of the DOM do not announce irrelevant decor.

Touch targets are implemented with a minimum size.

Logo Selection Based on Viewport Dimensions

A horizontally structured primary logo is implemented for "normal" width viewports and a compact vertically structured secondary logo is implemented for very narrow viewports. The logo versions are implemented with CSS and directly in the DOM so the most appropriate logo is selected if stylesheets do not load.

Component Focused Implementation

The application is divided into reusable components that can be overridden and extended for different versions of a component.

Pure CSS Dynamic Functionality

Dynamic functionality in the main navigation is implemented with pure CSS. Page identifiers in the DOM allow for current page menu item indicators. Descendant combinators and strategically disabling pointer events allows for emphasizing focused menu items and de-emphasizing sibling menu items. Sibling combinators and the checkbox checked pseudo class allows for an expandable “hamburger” menu for narrow screens.

Grid Layouts for Structure & Layering

Grid layouts with named grid areas are implemented for precise layouts and better visualization of layout structures when troubleshooting. Grid areas can also be redefined for responsive layouts without changing the grid items.

Grids allow for stacking elements by positioning multiple elements within a single grid area. Element dimensions are limited to a grid cell, avoiding the need to calculate the dimensions of absolutely positioned elements.

Responsive Sizing with Font Size & Aspect Ratio

Element sizes and media queries are based on the user's font size. All containers are implemented to break words if necessary to prevent horizontal scroll.

Containers with images have defined width and aspect ratio. The user agent calculates the height of the container based on the aspect ratio.

Printer Friendly Formatting

For print, simple flow layouts are implemented to prevent user agent specific print limitations. Property listings are displayed in tabular format for a concise, comparable view.
