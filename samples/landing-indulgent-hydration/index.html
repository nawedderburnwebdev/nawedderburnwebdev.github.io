Landing Page Slideshow

Landing page with multi-slide presentation layout for fictional company “Virna & Nuris”. Responsive components implemented with container queries that adjust to slideshow presentations in flow layouts, fixed dimension layouts, and preview thumbnails.

Description#

Web Design/Development

Halt Animations While Page Loads

A loader animation determined by the user's motion preference is implemented to prevent the intro slide animations from executing before the page loads. If JavaScript is enabled, a render and parser blocking script inserts an animated page loader into the DOM. Page elements other than the loader are hidden until the page loads or a maximum loader display time is exceeded. If the loader fails at any point, recovery error handlers remove the loader if it was created and the body elements are displayed.

Priority Loading Above the Fold

Give load priority to critical external assets on the first slide. Each critical asset is linked with a high fetchpriority in the DOM header.

Component Dimensions Determined by Parent Container

Container queries are used to configure component properties based on the dimensions of the parent container instead of the viewport. Properties can be adjusted based on the width of the parent or both width and height in a fixed dimension layout.

Fixed dimension containers are used to implement slide components that span the full viewport dimensions for a slideshow presentation. The slides can also be drastically reduced in size and displayed in a slideshow preview presentation.

Scroll Indicators for Scrollable Containers

Scroll indicators are implemented for devices with auto-hide scrollbars to indicate overflowing, scrollable content. The "scroll shadows" design pattern is used to form a composite background image using gradients to show and hide scroll indicator images.

Responsive & Configurable Slide Display

A slide is a configurable component. Default properties can be overridden for each slide implementation and unique layouts can be applied. The structure is implemented with container queries so its dimensions are determined by the dimensions of its container.

A slide is displayed in a flow layout by default. When slides are displayed in a slideshow with adequate viewport and container dimensions, the slides are displayed in a fixed dimension layout. With fixed slide dimensions, content overflow is controlled by clipping or scrolling.

Responsive, Accessible Slide Navigation

The display of the keyboard and screen reader accessible navigation is based on the slide display. For flow layout, the navigation container is sticky positioned, keeping the navigation in the vertical center of the viewport until it reaches the bottom of the slide container. For fixed dimension layout, slide navigation anchors are absolutely positioned on the bottom corners of the slide.

Slide Presentation Configuration

Slides can be displayed in a slideshow presentation or a slide preview presentation.

In a slideshow presentation, slides are displayed in a flow layout or fixed dimension layout based on the full dimensions of the viewport. If the presentation encompasses the full document, the page header and footer are incorporated into the slideshow display. Scroll snapping is implemented on the root element of pages that contain slide show presentations to allow slides to snap into place while scrolling.

In a slide preview presentation, slides are displayed as preview thumbnails. The presentation is configured as a grid with fixed dimension slides as the grid items. The responsive dimensions of the grid items can be defined for each presentation using custom properties for slide width and height.

Print with Screen Display or Minimal Formatting

Components can be printed unformatted or with the stylized formatting usually applicable only for screen display. This allows for printing slides as they appear on screen in the presentation preview layout.

Color Reversal for Dark Mode

All surfaces have light background colors and contrasting dark text. When user's preferred color scheme is dark, the colors are reversed such that the surfaces have dark background colors and the text is lighter in contrast.
